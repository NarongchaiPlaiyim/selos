W/C Requirement

 [View Change]
ความต้องการเงินทุนหมุนเวียน บาท รวมวงเงินสินเชื่อหมุนเวียนของผู้กู้และผู้เกี่ยวข้อง บาท ความต้องการทุนหมุนเวียนขาด (เกิน) บาท
รวมวงเงินสินเชื่อหมุนเวียนที่นำมาหัก บาท
กรณีที่ 1 : คำนวณจาก 1.25 เท่า ของ W/C requirement
วงเงิน บาท วงเงินขั้นต่ำที่สามารถพิจารณาให้ได้ บาท Core W/C 50% บาท วงเงิน OD หลังหัก Core W/C บาท

วงเงิน บาท วงเงินขั้นต่ำที่สามารถพิจารณาให้ได้ บาท Core W/C 50% บาท วงเงิน OD หลังหัก Core W/C บาท
